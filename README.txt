# RedPill
A Azure Cloud Serive for Redify Interview

You need to install windows azure tools to open this solution. It can be downloaded from http://azure.microsoft.com/en-us/downloads/
If you want to publish it you need to have a Azure subscription. 

This service is already published at http://myredpill.cloudapp.net/RedPill.svc (This will expire on 20th Feb 2015)

you can also download the client application from https://github.com/qianhaopower/RedPillClient to test this service.
